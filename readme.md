# livCricket

> livCricket is a simple CLI made using React-ink for cricket lovers.


## Install

```bash
$ git clone https://github.com/shakeabi/livCricket.git
$ npm install --global livCriket
```


## CLI

```
$ livCriket --help

  Usage
    $ livCriket

  Options
    --version

  Examples
    $ livCriket --version
    1.0.0
```
